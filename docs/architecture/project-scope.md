# Scope del Progetto

## In Scope (Iniziale)
- Sistema di autenticazione base (Supabase).
- Onboarding per la creazione di un singolo terreno.
- Dashboard con widget meteo (Open-Meteo API).
- Visualizzazione del terreno (Mappa concettuale/interattiva).
- Gestione lista piante (aggiunta/modifica/rimozione).
- Integrazione Arnaldo AI (Gemini API) per consigli contestuali.

## Out of Scope (Per ora)
- Gestione di terreni multipli.
- Integrazione con sensori IoT reali.
- E-commerce per prodotti agricoli.
- Social network tra agricoltori.
- Reportistica avanzata pluriennale.

## Sviluppo Incrementale
Il progetto DEVE essere costruito per step atomici. Ogni nuova feature deve essere preceduta da un'analisi dell'architettura esistente e non deve rompere i pattern già consolidati.
- **Fase 1**: Setup Documentale e Core Rules (In corso).
- **Fase 2**: Setup Infrastruttura (Next.js, Supabase, Auth).
- **Fase 3**: Onboarding e Data Modeling Terreno.
- **Fase 4**: Dashboard e Integrazione Meteo.
- **Fase 5**: Sistema Arnaldo e Mappa Piante.
