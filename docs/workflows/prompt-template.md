# Template per i Prompt Successivi

Copia e usa questo template per guidare Antigravity nei prossimi passi del progetto.

---

## Template Prompt

**Skills da usare**: @[/...], @[/...]

**Contesto da leggere**:
- GEMINI.md
- AGENTS.md
- docs/context/project-state.md
- docs/context/decisions.md

**Obiettivo Singolo**:
[Descrivi qui il task atomico, es: "Inizializza il progetto Next.js con Tailwind"]

**Vincoli**:
- Rispettare lo stack tecnologico definito.
- Seguire la design direction (Beige/Salvia).
- Non allargare lo scope.
- Naming in English per il codice, Italiano per i commenti e la UI.

**Output Atteso**:
1. Esecuzione dei comandi/creazione file.
2. Verifica del risultato.
3. Aggiornamento di `docs/context/project-state.md` con il nuovo stato.
4. Aggiornamento di `docs/context/decisions.md` se sono state fatte scelte tecniche.
5. Aggiornamento di `docs/context/backlog.md` (segna come completato).
